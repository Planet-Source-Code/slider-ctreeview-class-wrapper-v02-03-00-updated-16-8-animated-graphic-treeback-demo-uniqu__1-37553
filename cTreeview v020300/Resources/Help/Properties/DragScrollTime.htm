DragScrollTime Property

Returns/sets Auto Scroll timer delay (milliseconds) during Drag and Drop operation when the cursor is near the border of the Treeview control.

Syntax

object.DragScrollTime [=value]

The DragScrollTime property syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
value A value or constant that determines the delay time in milliseconds.

Remarks

The DragScrollTime property default setting is 200ms (0.2 second).

Data Type

Long
