NodeNestingLevel Property

Returns the branch depth/nesting level of the referenced Node object in the TreeView control .

Syntax

object.NodeNestingLevel(Node) = value

The NodeNestingLevel property syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
Node Required. A valid Node (MSComctlLib.Node) object.
value A return value that specifies the branch depth/nesting level.

Data Type

Long
