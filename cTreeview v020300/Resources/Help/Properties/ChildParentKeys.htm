ChildParentKeys Property

Returns a string array of Node.Keys for a specified node.

Syntax

object.ChildParentKeys(Node)

The ChildParentKeys property syntax has these parts:

Part Description
object An object expression that evaluates to an object in the Applies To list.
Node Required. A valid Node (MSComctlLib.Node) object.

Remarks

If the node passed is empty or invalid, an error occurs.

Data Type

Dynamic String Array
