Bold Property

Returns/sets the disabled Foreground colour in the Treeview control.

Syntax

object.Bold [=state]

The Bold property syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
state A value or constant that determines the disabled bold state, as described in Settings.

Settings

The Bold property settings are:

Setting Description
True All Disabled Node objects are set Bold..
False (Default) Bold is not set for selected Node objects.

Remarks

With the VB6 Treeview control, each Node object can have unique display properties (i.e. ForeColor, BackColor, Bold, etc..). These values are not lost when using custom disabled properties.

When the Bold property is changed, all disabled Node objects are changed automatically. 

Data Type

Boolean
