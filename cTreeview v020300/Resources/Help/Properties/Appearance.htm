Appearance Property

Returns/sets whether or not the Treeview control has a 3D (Fixed single), Thin 3D or none.

Syntax

object.Appearance [=value]

The Appearance property syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
value A value or constant that determines the border style, as described in Settings.

Settings

The Appearance property settings are:

Constant Setting Description
[None] 0 None (No border).
[Fixed Single] 1 Fixed. The control appears with a 3D border.
[Thin] 2 Thin. The control apears with a light 3D border.

Remarks

If the Appearance property is set to [Fixed Single], the control appears with the standard 3D sunken border.

Data Type

eAppearance
