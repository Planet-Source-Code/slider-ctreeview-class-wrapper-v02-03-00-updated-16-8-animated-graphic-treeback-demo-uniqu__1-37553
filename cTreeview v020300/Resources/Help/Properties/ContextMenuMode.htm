ContextMenuMode Property

Sets whether the ContextMenu Event fires before or after the Treeview control KeyPress or Click Events.

Syntax

object.ContextMenuMode [=value]

The ContextMenuMode property syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
value A value or constant that determines when the ContextMenu Event fires, as described in Settings.

Settings

The ContextMenuMode property settings are:

Constant Setting Description
[Before Click] 0 ContextMenu event fires before Treeview KeyPress or Click Events.
[After Click] 1 ContextMenu event fires before Treeview KeyPress or Click Events.

Remarks

The ContextMenuMode property default setting is [After Click].

Data Type

eContextMenuMode
