Ctrl Property

Exposes the Treeview control's properties and methods that are not wrapped by the cTreeview wrapper class.

Syntax

object.Ctrl

The Ctrl property syntax has these parts:

Part Description
object An object expression that evaluates to an object in the Applies To list.

Data Type

MSComctlLib.Treeview

Example

Dim oTree as vbTree.cTreeview
set oTree = new vbTree.cTreeview

With oTree
    .Hook Treeview1
    With .Ctrl
        .Style = tvwTreelinesPlusMinusPictureText
        .Linestyle = tvwRootlines
        .Indentation = 10
        .FullRowSelect = False
        .HideSelection = False
        .HotTracking = True
        .LabelEdit = tvwAutomatic
    End With
End With
