ClearSelection Method

De-selects all selected Node objects in the referenced Treeview control restoring each Node object's Foreground, Background and Bold properties.

Syntax

object.ClearSelection

The object placeholder represents an object expression that evaluates to an object in the Applies To list.

Remarks

After the selection is cleared, the NoDefaultSel property determines whether the last selected node is highlighted and focus rectangle is drawn or only the focus rectangle.
