SelectMode Property

Returns/sets how a selected Node object is handled when disabled.

Syntax

object.SelectMode[=state]

The SelectMode property syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
State A value or constant that determines the how a selected Node object is handled when disabled, as described in Settings.

Settings

The SelectMode property settings are:

Setting Description
[eiDis Select Direction] Find next enabled Node object.
[eiDis Focus Rectangle] (Default) Show focus rectangle over disabled node.
[eiDis Select None] Set selection to nothing. 

Remarks

Data Type

eDisableAdjustSelectMode
