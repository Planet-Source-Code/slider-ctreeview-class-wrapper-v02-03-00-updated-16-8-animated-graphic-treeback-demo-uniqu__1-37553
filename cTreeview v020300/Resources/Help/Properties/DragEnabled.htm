DragEnabled Property

Enables/Disables Drag and Drop operation. (NOTE: Cannot be used with MutliSelect feature!)

Syntax

object.DragEnabled [=value]

The DragEnabled property syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
value A value or constant that determines whether Drag and Drop is allowed, as described in Settings.

Settings

The DragEnabled property settings are:

Setting Description
True Drag and Drop of Nodes is allowed.
False Drag and Drop of Nodes is disabled.

Remarks

The DragEnabled property default setting is True.

Data Type

Boolean
