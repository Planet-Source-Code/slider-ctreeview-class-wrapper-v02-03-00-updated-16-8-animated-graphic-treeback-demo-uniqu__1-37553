BackFileName Property

Returns/sets the custom Treeview control background image Filename.

Syntax

object.BackFileName [=value]

The BackFileName property syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
value Required. String expression indicating the name of the file to be loaded.

Remarks

Be sure to pass a valid image filename otherwise an error will be raised. When the Filename is passed and object.BackMode = vbGraphic, the image will be automatically loaded and displayed.

Data Type

String 

Example

'
'## Note: oTree is a reference to the vbTree.cTreeView class
'
'## Load Graphic Background image from file
'
With oTree
    .Backmode = bmGraphic
    .BackFileName = App.Path + "\TreeBack.Gif"
End With
