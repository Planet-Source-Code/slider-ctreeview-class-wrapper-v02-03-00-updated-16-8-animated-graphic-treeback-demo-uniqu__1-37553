ForeColor Property

Returns/sets the disabled Foreground colour in the Treeview control.

Syntax

object.ForeColor [=color]

The ForeColor property syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
color A numeric expression that specifies the colour.

Remarks

With the VB6 Treeview control, each Node object can have unique display properties (i.e. ForeColor, BackColor, Bold, etc..). These values are not lost when using custom disabled node properties.

When the ForeColor property is changed, all disabled Node objects are changed automatically. 

Default setting is VBRUN.SystemColorConstants.vbGrayText

Data Type

stdole.OLE_COLOR
