Locked Property

Returns or sets a value indicating whether the Treeview control can be edited with true Locked appearance.

Syntax

object.Locked [=value]

The Locked property syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
value A boolean expression that specifies whether the TreeView control can be edited, as described in Settings.

Settings

The Locked property settings are:

Setting Description
True You can scroll and select nodes in the control, but you can't edit it plus 'locked' rendering and state is set.
False (Default) You can edit the nodes in the control and the 'normal' rendering and state is set.

Data Type

Boolean
