SelectedNode Property

Returns a specific Node object of the internal cTreeView wrapper Dll/Class SelectedNodes collection, either by index or by key.

Syntax

object.SelectedNode(index)

The SelectedNode property syntax the following parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
index Required. An expression that specifies the index of a Node in the internal cTreeView wrapper Dll/Class SelectedNodes collection.

If a numeric expression, index must be a number from 1 to the value of the Count property.

If a string expression, index must correspond to the Key property of the Node.

Remarks

If the value provided as index does not match ant existing member of the internal cTreeView wrapper Dll/Class SelectedNodes collection, an error occurs.
