Enabled Property

Returns/sets a value that determines whether the TreeView control can respond to user-generated events with true disabled appearance.

Syntax

object.Enabled [=value]

The Enabled property syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
value A boolean expression that specifies whether the TreeView control can respond to user-generated events, as described in Settings.

Settings

The Enabled property settings are:

Setting Description
True (Default) The control will respond to user-generated events.
False Prevents control from responding to events.

Data Type

Boolean
