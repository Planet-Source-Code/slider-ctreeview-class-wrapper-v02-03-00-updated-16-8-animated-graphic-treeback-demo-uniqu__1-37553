AllowSelect Property

Returns/sets the whether disabled Node objects can be selected in the Treeview control.

Syntax

object.AllowSelect[=state]

The AllowSelect property syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
state A value or constant that determines the selection of disabled Node objects, as described in Settings.

Settings

The AllowSelect property settings are:

Setting Description
True Enable selection of disabled Node objects.
False (Default) Disable selection of disabled Node objects.

Remarks

Data Type

Boolean
