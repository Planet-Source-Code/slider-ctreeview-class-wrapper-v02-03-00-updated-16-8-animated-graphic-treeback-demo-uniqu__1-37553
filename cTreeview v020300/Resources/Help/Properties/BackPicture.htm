BackPicture Property

Returns/sets the custom Treeview control background image from a referenced StdPicture object.

Syntax

object.BackPicture [=value]

The BackPicture property syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
value Required. StdPicture object.

Remarks

When the BackPicture is set and object.BackMode = vbGraphic, the image will be automatically displayed.

Data Type

StdPicture

Example

'
'## Note: oTree is a reference to the vbTree.cTreeView class
'
'## Set Graphic Background image from a StdPicture object reference. In
'   this example, we're referencing a PictureBox control
'
With oTree
    .Backmode = bmGraphic
    Set .BackPicture = Picture1.Picture 
End With

 

'
'## Note: oTree is a reference to the vbTree.cTreeView class
'
'## Set Graphic Background image from a StdPicture object reference. In
'   this example, we're referencing an ImageList Picture
'
With oTree
    .Backmode = bmGraphic
    Set .BackPicture = ImageList1.ListImages("TreeBack").Picture
End With
