CutIcon Property

Returns/sets the referenced Node object's cut icon state.

Syntax

object.CutIcon(Node) [=value]

The CutIcon property syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
Node Optional. A valid Node (MSComctlLib.Node) object. If no node passed, then the selected Node will be used.
value A value or constant that determines the cut icon state, as described in Settings.

Settings

The CutIcon property settings are:

Setting Description
True Renders and flags the Node's Icon as 'Cut' state like MS Explorer.
False Renders and flags the Node's Icon as 'Normal' state.

Data Type

Boolean
