02.03.00 Addition New Feature: TreeView BackColor and BackPicture! There have been other versions of BackPicture available like Steve McMahon's cTVBackGround but they all suffered from the same problem - individual node colors were not supported. Therefore all nodes had their Forecolor property set to black and the Backcolor set to transparent. Also, there have been hacks around for setting the BackColor of the TreeView control and they all had strange side-effects...

The solution is here!

 

02.03.00 Addition New Feature: Disabled Nodes Interface now integrated with Multi Node Selection. 
02.03.00 Addition New Tutorial: Tutorial 8 illustrates the features of the Multi Node Selection with Disabled Nodes.
02.03.00 Addition New Demo: Demo 1 was put together to show off the new BackPicture feature. PSC (Planet Source Code) has many users that want fancy graphics - this is my contribution to answer my critiques - enjoy guys!
02.03.00 Update Tutorial 1: Added new BackMode  features and cFileDialog class (by Klaus H. Probst) for Opening/saving files. 
02.02.00 Addition New Feature: Disabled Nodes Interface with full keyboard and mouse support - Allows individual nodes to be enabled/disabled (not selectable) and toggled with the ability to affect child nodes with a single command. Disabled Fore/Back colours and Bold state are programmable - preserves the original Forecolor, Backcolor and bold status of each individual node. StateChange event fires when a node or a range of nodes are disabled/enabled. 7 properties & 4 methods - uses same design philosophy as Multi Node Selection.

Multi Node Selection currently does not support Disabled Nodes interface.

 
02.02.00 Addition All Tutorials that have TabStrip control(s) now utilize the cTabStrip Wrapper class.
02.02.00 Addition New Tutorial: Tutorial 6 illustrates the three types of Node find methods:- Find a Specific Node (exact match case-insensitive), Find a Group of Nodes (containing), Find a Group of Nodes (Text/Binary Pattern Matching). I've also included benchmarking to help indicate which method is the quickest. 

[Note: If the tutorial's pInitData subroutine is modified to load a Treeview-data file from your application (saved using the cTreeView.Save method), then this tutorial could be used to decide which search method is the quickest for your application!]
02.02.00 Addition New Tutorial: Tutorial 7 illustrates the features of the Disabled Nodes Interface.
02.02.00 Addition 13 New Methods: HitTestEx, IsNodeTextUnique, IsPathExpanded,  NodeAbsolutePosition, NodeAbsoluteIndex, NodeCoord, NodeCountChildren, NodeFindEx1, NodeFindEx2, NodeFindObjectInTag, NodePosition, NodeUniqueText, SubNodesSorted.
