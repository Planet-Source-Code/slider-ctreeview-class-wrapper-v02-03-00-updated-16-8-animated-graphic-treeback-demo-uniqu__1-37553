SubNodesSorted Method

Sets the sort properties of the specified Node object and for all child Node objects for all sub-branch levels of the Treeview control.

Syntax

object.SubNodesSorted(Node, State)

The SubNodesSorted method syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
Node Required. A valid Node (MSComctlLib.Node) object. If an empty Node object is passed, then all Nodes of the TreeView control's will sorted property will be set.
State Required. A Boolean value or constant that specifies whether Node objects are sorted, as described in settings.

Settings

The settings for Sate operator are:

Constant Setting Description
True -1 (Default) Enable node sorting.
False 0 Disable node sorting.
