NodeFind Method

Simple search for a specific Node object in the TreeView control.

Syntax

object.NodeFind(oResultNode, NodeText, NodeKey, bSelect)

The NodeFind Method syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
oResultNode Required. A valid Node (MSComctlLib.Node) object. Result of Find operation.
NodeText Required. Node Text to find.
NodeKey Optional. Node Key to find.
bSelect Optional. Set selection to oResultNode if Find operation was successful.

Settings

The settings for NodeFind Method are:

Setting Description
True Operation was successful.
False Operation failed. (Trap the Err object for error condition)

Remarks

This method is more like an Exist routine than to search for a node in the Nodes object collection.
