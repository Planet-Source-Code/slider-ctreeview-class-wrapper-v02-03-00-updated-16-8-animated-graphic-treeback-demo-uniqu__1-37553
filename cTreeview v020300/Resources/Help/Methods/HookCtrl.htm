HookCtrl Method

This method is used to initialise the cTreeView wrapper Dll/class.

Syntax

object.HookCtrl(Ctrl, OpMode)

The HookCtrl method syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
Ctrl Required. A valid Treeview (MSComctlLib.Treeview) object.
OpMode Optional. A value or constant that specifies the operation mode of the cTreeView wrapper Dll/Class, as described in settings.

Settings

The settings for OpMode method are:

Constant Setting Description
[Single Select] 0 (Default) Only Single node selection allowed, Drag and Drop enabled.
[Multi Select] 1 Enables Multi-node Selection capabilities but disables Drag and Drop.

Remarks

If [Multi Select] mode is set, then the iMultiSelect interface must be implemented to access Multi-Node Selection functionality. When working with interface classes, all properties and methods must be used or an error will occur (See VB6 documentation for more information on working with interfaces).
