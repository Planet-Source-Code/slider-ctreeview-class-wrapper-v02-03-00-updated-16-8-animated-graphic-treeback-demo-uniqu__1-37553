SetOverlayImage Method

Used to Initialise Overlay Icon Images. Must be used before setting the Node's OverlayIcon state image.

Syntax

object.SetOverlayImage(Imaglelisthandle, ImageListIndex, ShareOverlay)

The SetOverlayImage Method syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
Imaglelisthandle Required. An ImageList control handle that's associated with the referenced Treeview control.
ImageListIndex Required. The index of an image in the associated ImageList control (linked to the referenced Treeview control).
ShareOverlay Required. A value or constant that specifies the overlay icon index, as described in settings.

Settings

The settings for ShareOverlay operator are:

Constant Setting Description
[No Overlay] 0 None.
[Share Overlay] 1 Share Overlay icon image..
[Shortcut Overlay] 2 Shortcut Overlay icon image.
[Custom 1] 3 Custom Overlay Image 1.
[Custom 2] 4 Custom Overlay Image 2.
[Custom 3] 5 Custom Overlay Image 3.
