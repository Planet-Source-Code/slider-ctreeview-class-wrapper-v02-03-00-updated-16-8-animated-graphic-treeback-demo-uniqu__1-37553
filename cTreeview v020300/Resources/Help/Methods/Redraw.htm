Redraw Method

Enable/Disable rendering of the referenced TreeView control.

Syntax

object.Redraw [=value]

The Redraw method syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
value A boolean expression that specifies whether the TreeView control renders, as described in Settings.

Settings

The Redraw Method settings are:

Setting Description
True (Default) The control will render the view area of the referenced Treeview control.
False Disable the rendering of the view area of the referenced Treeview control.

Remark

This routine does not work for all functions. Code left in class for backward compatibility. The alternative method is to toggle the Visible property (and most reliable to speed up functionality that requires the Treeview control to be rendered) of the Treeview control.

Data Type

Boolean
