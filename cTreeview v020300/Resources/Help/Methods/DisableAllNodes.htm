DisableAllNodes Method

Disables all Node objects in the referenced Treeview control or disables all Node objects from a designated parent Node Object.

Syntax

object.DisableAllNodes ( Node, State )

The DisableAllNodes method syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
Node Optional. A valid Node (MSComctlLib.Node) object.
State Optional. A boolean expression that specifies whether the node is enabled or disabled, as described in Settings.

Settings

The State operator settings are:

Setting Description
True (Default) The Node object(s) disabled.
False The Node object(s) enabled.
