SelectNode Method

Set the Selected status of a specific Node object in the referenced Treeview control.

Syntax

object.SelectNode ( Node, Selected )

The SelectNode method syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
Node Required. A valid Node (MSComctlLib.Node) object.
Selected Required. A boolean that determines the selection state of the referenced Node object.

Settings

The settings for Selected are:

Setting Description
True The referenced Node object is selected.
False The referenced Node object is not selected
