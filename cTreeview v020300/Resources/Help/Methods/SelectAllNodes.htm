SelectAllNodes Method

Selects all Node objects in the referenced Treeview control or selects all Node objects from a designated parent Node Object.

Syntax

object.SelectAllNodes ( Node )

The SelectAllNodes method syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
Node Optional. A valid Node (MSComctlLib.Node) object.
