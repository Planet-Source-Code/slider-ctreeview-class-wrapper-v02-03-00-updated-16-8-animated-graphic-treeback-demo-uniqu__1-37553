NodeAbsoluteIndex Method

Returns a Node objects absolute Index in the TreeView control regardless of Add, Move, Delete, Expanded or Collapsed operations.

Syntax

object.NodeAbsoluteIndex(Node, Text, CompareMethod )

The NodeAbsoluteIndex Method syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
Node Required. A valid Node (MSComctlLib.Node) object. If an empty Node object is passed, then an error will be raised.

(See also NodeAbsolutePosition)
