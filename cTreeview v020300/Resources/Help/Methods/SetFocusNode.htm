SetFocusNode Method

Shows the focus rectangle without actually selecting the Node object.

Syntax

object.SetFocusNode(index)

The SetFocusNode Method syntax the following parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
index Required. An expression that specifies the index of a Node in the internal cTreeView wrapper Dll/Class SetFocusNodes collection.

If a numeric expression, index must be a number from 1 to the value of the Count Method.

If a string expression, index must correspond to the Key Method of the Node.
