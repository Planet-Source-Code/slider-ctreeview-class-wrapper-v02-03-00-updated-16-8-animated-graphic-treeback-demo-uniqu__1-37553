NodeUniqueText Method

Returns unique text based on parameters passed.

Syntax

object.NodeUniqueText(Node, NewText, CompareMethod, PrefixNumberMode, Prefix )

The NodeUniqueText Method syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
Node Required. A valid Node (MSComctlLib.Node) object. If an empty Node object is passed, then all of the TreeView control's nodes will be tested.
Text Required. The text to test and make unique.
CompareMethod Optional. Specifies the type of string comparison, as described in settings. If the compare argument is Null, an error occurs.
PrefixNumberMode Optional. Method of making Text unique, as described in settings.
Prefix Optional. Prefix text (Default = 'Copy of ') if PrefixNumberMode is [PreFix].

Settings

The settings for CompareMethod operator are:

Constant Setting Description
vbBinaryCompare 0 Performs a binary comparison.
vbTextCompare 1 (Default) Performs a textual comparison.

The settings for PrefixNumberMode operator are:

Constant Setting Description
[Numbers] 0 (Default) Adds braced number to end of Text to generate unique Text.
[Prefix] 1 Prefixes Prefix text to Text to generate unique Text.

The settings for NodeUniqueText method are:

Setting Description
True The Text is unique.
False A Node object already has Text.

Remarks

Data Type

String
