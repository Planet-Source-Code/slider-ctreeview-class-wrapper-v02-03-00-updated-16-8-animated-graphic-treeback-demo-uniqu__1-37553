Load Method

Loads the referenced Treeview control from a specified binary or Xml file.

Syntax

object.Load(FileName, FileMode)

The Load Method syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
FileName Required. A valid file path and filename.
FileMode Optional. A value or constant that specifies the type of file being loaded, as described in settings

Settings

The settings for Load Method are:

Setting Description
True Operation successful.
False Operation failed. (Trap the Err object for error condition)

The settings for FileMode operator are:

Constant Setting Description
[Binary] 0 (Default) Load from a binary file.
[Xml] 1 Load from an Xml file..

Remarks

This operation will Clear the referenced Treeview control before loading from file.
