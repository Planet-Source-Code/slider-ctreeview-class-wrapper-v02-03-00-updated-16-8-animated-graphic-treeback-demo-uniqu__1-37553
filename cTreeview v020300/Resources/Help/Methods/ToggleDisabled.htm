ToggleDisabled Method

Toggles the enabled/disabled state of a specific Node object in the referenced Treeview control.

Syntax

object.ToggleDisabled ( Node)

The ToggleDisabled method syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
Node Required. A valid Node (MSComctlLib.Node) object.
