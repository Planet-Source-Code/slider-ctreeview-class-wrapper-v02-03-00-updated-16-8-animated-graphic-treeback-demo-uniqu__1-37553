DisableMultiSelect Method

Set the Disabled status of a specific Node object in the referenced Treeview control.

Syntax

object.DisableMultiSelect ( State )

The DisableMultiSelect method syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
State Optional. A Boolean that determines the disabled state of the Multi Select Nodes collection, as described in Settings.

Settings

The State operator settings are:

Setting Description
True (Default) The Node objects in the Multi Select collection are disabled.
False The Node objects in the Multi Select collection are enabled.
