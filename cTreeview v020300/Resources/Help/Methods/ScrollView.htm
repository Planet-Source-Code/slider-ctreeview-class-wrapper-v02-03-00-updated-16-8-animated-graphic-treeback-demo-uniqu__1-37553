ScrollView Method

Scroll the viewabe area of the TreeView control in all four directions.

Syntax

object.ScrollView(Dir)

The ScrollView method syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
Dir Required. A value or constant that specifies the direction to scroll the view window, as described in settings.

Settings

The settings for WhichScroll operator are:

Constant Setting Description
[Home] 0 Scroll to top.
[Page Up] 1 Page up.
[Up] 2 Line up.
[Down] 3 Line down.
[Page Down] 4 Page down.
[End] 5 Scroll to bottom.
[Left] 6 Scroll to far left.
[Page Left] 7 Page left.
[Line Left] 8 Line left.
[Line Right] 9 Line right.
[Page Right] 10 Page right.
[Right] 11 Scroll to far right.
