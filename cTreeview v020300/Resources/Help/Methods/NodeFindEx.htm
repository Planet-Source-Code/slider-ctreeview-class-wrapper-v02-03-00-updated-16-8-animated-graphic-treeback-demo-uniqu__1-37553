NodeFindEx Method

Search for a specific Node object &/or a Group of Node objects containing specific text in the TreeView control.

Syntax

object.NodeFindEx(Text, Mode, CaseSensitive)

The NodeFindEx Method syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
Text Required. partial text to find. 
Mode Optional. Specifies what property of the Node object to search, as described in settings.
CaseSensitive Optional. Specifies whether to force case matching, as described in settings.

Settings

The settings for Mode operator are:

Setting Description
[Find Node Text] (Default) Only searches a Node object's Text property.
[Find Node Key] Only searches a Node object's Key property.
[Find Node Text And Key] Search the Node object's Text & Key properties.

The settings for CaseSensitive operator are:

Setting Description
True Case-Sensitive.
False (Default) Case-Insensitive

Remarks

No wild cards supported.

Example: *ID* will search for a Node.Text containing *ID*. Therefore if the node you are searching for is Book ID 3, then no matches will be found. The correct method is to search for ID.
