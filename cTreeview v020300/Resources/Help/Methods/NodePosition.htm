NodePosition Method

Returns the ordinal position of a node object with it's own sibling Node objects in the TreeView control.

Syntax

object.NodePosition(Node)

The NodePosition Method syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
Node Required. A valid Node (MSComctlLib.Node) object. If an empty Node object is passed, then an error will be raised.
