NodeAbsolutePosition Method

Returns a Node objects absolute position in the TreeView control dependant on Node object visibility.

Syntax

object.NodeAbsolutePosition(Node, VisibleOnly )

The NodeAbsolutePosition Method syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
Node Required. A valid Node (MSComctlLib.Node) object. If an empty Node object is passed, then an error will be raised.
VisibleOnly Optional. A valid Boolean value, as described in settings.

Settings

The settings for VisibleOnly method are:

Setting Description
True (Default) Only count Node objects that are not hidden by a collapsed parent Node object.
False Count all Node objects regardless of their parent Node object Expand property setting.
