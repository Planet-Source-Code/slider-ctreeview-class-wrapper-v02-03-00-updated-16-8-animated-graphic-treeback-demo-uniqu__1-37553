NodeCoord Method

Returns the selected Node object coordinates in the TreeView control.

Syntax

object.NodeCoord(Left, Top, Width, Height, HostType )

The NodeCoord Method syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
Left Required. Returned single-precision value indicating the horizontal coordinate (x-axis) for the left edge of the selected Node object.
Top Required. Returned single-precision value indicating the vertical coordinate (y-axis) for the top edge of the selected Node object.
Width Required. Returned single-precision value indicating the new width of the selected Node object.
Height Required. Returned single-precision value indicating the new height of the selected Node object.
HostType  Optional. Specifies how to calculate the returned Width value.

Settings

The settings for HostType method are:

Constant Setting Description
[By Node] 0 Width is the width of the TreeView control's Editbox.
[By TreeView] 1 Width returned is calculated to the edge of the TreeView control's viewport.
