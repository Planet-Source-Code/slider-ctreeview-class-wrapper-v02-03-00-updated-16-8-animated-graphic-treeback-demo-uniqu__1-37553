IsScrollbarVisible Method

Returns True if the specified scrollbar(s) is/are visible in the TreeView control.

Syntax

object.IsScrollbarVisible(WhichScroll)

The IsScrollbarVisible Method syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
WhichScroll Required. A value or constant that specifies which scrollbars to test for, as described in settings.

Settings

The settings for IsScrollbarVisible Method are:

Setting Description
True Scrollbar(s) are visible.
False Scrollbar(s) are not visible.

The settings for WhichScroll operator are:

Constant Setting Description
[Horizontal] 1 Test for Horizontal scrollbar visibility.
[Vertical] 2 Test for Vertical scrollbar visibility.
