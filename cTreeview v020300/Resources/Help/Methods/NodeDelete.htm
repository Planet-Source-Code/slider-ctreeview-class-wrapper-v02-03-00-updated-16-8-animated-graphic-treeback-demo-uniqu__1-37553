NodeDelete Method

Copies a Node object and optionally it's child nodes from one parent  Node object to another  Node object in the TreeView control.

Syntax

object.NodeDelete(Node, bSelect, bIncludeChildren)

The NodeDelete Method syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
Node Required. A valid Node (MSComctlLib.Node) object.
bSelect Optional. Set selection to parent of Node after delete operation.
bIncludeChildren Optional. A boolean value or constant that specifies whether to include child Node objects in the copy operation, as described in settings.

Settings

The settings for NodeDelete Method are:

Setting Description
True Operation was successful.
False Operation failed. (Trap the Err object for error condition)

The settings for bIncludeChildren operator are:

Setting Description
True delete all child Node objects that are associated with Node.
False delete only the Node.
