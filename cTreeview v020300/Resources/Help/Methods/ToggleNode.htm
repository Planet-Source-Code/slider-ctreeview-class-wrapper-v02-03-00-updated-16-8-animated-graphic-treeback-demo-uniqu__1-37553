ToggleNode Method

Toggles the selection status of a specific Node object in the referenced Treeview control or toggles the selection status for all Node objects from the designated parent Node Object.

Syntax

object.ToggleNode ( Node, Children )

The ToggleNode method syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
Node Optional. A valid Node (MSComctlLib.Node) object.
Children Optional. A boolean value that specifies the toggling of selected child Node objects.

Settings

The settings for Children are:

Setting Description
True All child Node object selectestions are toggled.
False Only the specified Node object or focus Node object selection status is toggled.

Remarks

If no Node object is passed, then the focus Node object selection status is toggled.
