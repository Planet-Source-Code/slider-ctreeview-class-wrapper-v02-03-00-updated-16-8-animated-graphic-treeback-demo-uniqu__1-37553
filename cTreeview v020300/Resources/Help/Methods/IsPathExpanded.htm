IsPathExpanded Method

Returns if a Node object's path is expanded, back to the Root-level Node object, in the TreeView control.

Syntax

object.IsPathExpanded(Node )

The IsPathExpanded Method syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
Node Required. A valid Node (MSComctlLib.Node) object.

Settings

The settings for IsPathExpanded Method are:

Setting Description
True The Node object's full path is expanded.
False The Node object's full path is not expanded.
